# ⏱️ TimeTracker - 简洁计时器

一个简洁的 Android 计时记录应用，支持毫秒级计时、统计分析和数据导出。

## 📱 功能特性

- ✅ **毫秒级计时** - 精确到 0.001 秒
- ✅ **等宽字体显示** - 数字变化不抖动
- ✅ **统计分析** - 最短/最长/平均/总计时间
- ✅ **一键清理** - 快速清空记录
- ✅ **覆盖安装** - 签名一致，支持升级
